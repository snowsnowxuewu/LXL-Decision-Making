# 李笑来《思考的真相》决策模板

> 适用于任何重大决定。先完成「信息收集」，再按五步走。

---

## 第零步：信息收集（开始前必须回答）

在做任何分析之前，先回答以下问题。答案越具体，后面的分析越有价值。

### 必答问题

| # | 问题 | 你的回答 |
|---|------|---------|
| 1 | **这个决定是谁的决定？** 是你替别人决定，还是当事人自己决定？ | |
| 2 | **真正的目标是什么？** 不是"做什么"，而是"为了得到什么结果"？ | |
| 3 | **这个决定的动机从哪里来？** 是自己想清楚的，还是受别人影响的？ | |
| 4 | **最担心的是什么？** | |
| 5 | **最在意的是什么？** 如果只能保一件事，是哪件事？ | |
| 6 | **有没有时间限制？** 什么时候必须决定？ | |
| 7 | **有没有资源限制？** 钱、时间、精力是否有上限？ | |

### 示例（孩子去美国读大学）

| # | 问题 | 回答 |
|---|------|------|
| 1 | 谁的决定？ | 父母主导，孩子有意愿 |
| 2 | 真正的目标？ | 孩子毕业后能找到有意义且有竞争力的工作 |
| 3 | 动机来源？ | 孩子的同学想去美国，孩子跟着觉得"机会更多" |
| 4 | 最担心？ | 花了很多钱但结果不比加拿大好 |
| 5 | 最在意？ | 孩子的长期发展和幸福感 |
| 6 | 时间限制？ | 孩子16岁，约2年后申请 |
| 7 | 资源限制？ | 经济上有一定压力，美加差价约$300K USD |

## 第一步：定义

> 核心：把模糊的概念变成清晰的等号。
> 要求：**必要**（没用的删掉）+ **无歧义**（有歧义就反复更新）

### 操作方法

**找出所有模糊的词，逐一替换成清晰的描述：**

```
模糊表述：[你最初说的那句话]
              ↓
清晰定义：相比[现有选项A]，选择[选项B]，
          是否能更有效地帮助[谁]，
          在[时间范围]内，
          实现[具体且无歧义的目标]。
```

### 警示：这些词通常需要重新定义

- "更好的" → 更好在哪里？对谁更好？
- "更多机会" → 什么机会？在哪里？
- "好工作" → 什么行业？在哪里工作？什么收入水平？
- "更好的发展" → 什么维度的发展？

### 示例

```
模糊表述：让孩子去美国读大学，因为机会更多，将来能找到好工作。

清晰定义：相比继续留在加拿大完成大学教育，选择去美国读大学，
          是否能更有效地帮助一个对艺术/心理/人文感兴趣的16岁孩子，
          在毕业后（约6年内），
          获得有意义且有竞争力的职业发展——无论在哪个国家。
```

### 定义完成后问自己

- [ ] 所有关键词都清晰了吗？
- [ ] 这个定义是当事人自己的，还是借用了别人的？
- [ ] 定义里的目标是真正的目标，还是某个路径被误当成了目标？

---

## 第二步：分类

> 核心：把所有影响因素列出来并分类。
> 要求：**合理**（各类别不重叠）+ **完整**（不遗漏重要类别）

### 操作方法

**分类框架（适用于大多数人生决策）：**

| 大类 | 细分因素 | 是否适用 | 备注 |
|------|---------|---------|------|
| **当事人本身** | 能力、意愿、性格、适应力 | | |
| **目标达成路径** | 这个选项真的能达到目标吗？有没有其他路径？ | | |
| **资源条件** | 经济、时间、精力、人脉 | | |
| **外部风险** | 政策、环境、不可控因素 | | |
| **机会成本** | 选了这个，放弃了什么？ | | |
| **时间因素** | 短期/中期/长期影响是否一致？ | | |

### 完整性检查

列完之后问自己：**有没有哪个类别我根本没想到？**
（通常被遗漏的是：机会成本、当事人真实意愿、时间维度）

### 示例

| 大类 | 细分因素 |
|------|---------|
| 当事人 | 16岁，IB在读，兴趣在艺术/心理/人文，动机部分来自从众 |
| 目标路径 | 美国顶尖文理学院在人文强；但加拿大McGill/UBC同样强；心理学无论哪里读本科都需要读研 |
| 资源条件 | 美国比加拿大贵约$300K USD；孩子已适应加拿大环境 |
| 外部风险 | 美国H1B签证抽签制度，留美工作难度高；加拿大PR路径更清晰 |
| 机会成本 | 留加拿大可同时积累PR；去美国4年后需重新选择落脚点 |
| 时间因素 | 短期：适应新环境成本；长期：毕业后落脚选择变窄 |

## 第三步：因果

> 核心：验证"这个选项真的能带来想要的结果吗"。
> 工具：四问模板 + 相关≠因果检查

### 四问模板

```
① [选项A] 真的能带来 [目标结果] 吗？
   → 因果链是否成立？有没有证据？

② [选项A] 是实现 [目标结果] 的唯一方式吗？
   → 有没有其他路径也能达到同样目标？

③ 还有哪些因素会影响 [目标结果]？
   → 比[选项A]更重要的因素是什么？

④ 影响 [目标结果] 最重要的因素到底是什么？
   → 这才是决策的真正核心。
```

### 相关≠因果 检查

问自己：**我以为的因果，有没有可能只是相关？**

常见陷阱：
- "成功人士都读名校" → 因果还是相关？读名校导致成功，还是有能力的人更容易进名校？
- "同学去美国后发展很好" → 是美国的功劳，还是那个同学本来就很强？

### 示例

```
① 去美国读大学，真的能让孩子找到更好的工作吗？
   → 在艺术/心理/人文领域，美国顶尖校确实有优势；
     但"更好工作"在这些领域更依赖个人能力、作品集、读研院校，
     而非本科在哪个国家。因果链不够强。

② 去美国是唯一路径吗？
   → 不是。加拿大McGill/UBC/多伦多在人文同样世界级；
     英国、香港也是选项。

③ 还有哪些因素影响"好工作"？
   → 心理学方向：研究生院校比本科更重要
   → 艺术方向：作品集和实践经历比学校名气更重要
   → 孩子的学习动力和适应能力，比地点更重要

④ 最重要的因素是什么？
   → 孩子对自己方向的清晰度 + 选择一个能激发他/她潜力的环境
   → 这两点不必然指向"美国"
```

## 第四步：比较

> 核心：用客观标准打分，得出可比较的结论。
> 要求：判断依据客观、权重反映真实优先级。

### 操作方法

**第一步：列出判断依据（来自第二步的分类）**
**第二步：按重要性分配权重（加起来=100%）**
**第三步：给每个选项打分（1-10分）**
**第四步：加权总分 = 每项得分 × 权重，求和**

### 比较矩阵模板

| 判断依据 | 权重 | 选项A得分 | 选项A加权 | 选项B得分 | 选项B加权 |
|---------|------|---------|---------|---------|---------|
| [依据1] | % | /10 | | /10 | |
| [依据2] | % | /10 | | /10 | |
| [依据3] | % | /10 | | /10 | |
| [依据4] | % | /10 | | /10 | |
| [依据5] | % | /10 | | /10 | |
| **总分** | 100% | | **?** | | **?** |

### 示例（去美国 vs 留加拿大）

| 判断依据 | 权重 | 美国 | 美国加权 | 加拿大 | 加拿大加权 |
|---------|------|------|---------|-------|---------|
| 人文/心理领域教育质量 | 30% | 8 | 2.4 | 7 | 2.1 |
| 毕业后职业机会（含签证） | 25% | 5 | 1.25 | 7 | 1.75 |
| 经济成本（越低越好） | 20% | 3 | 0.6 | 8 | 1.6 |
| 孩子适应性与幸福感 | 15% | 6 | 0.9 | 8 | 1.2 |
| 未来选择灵活性 | 10% | 5 | 0.5 | 7 | 0.7 |
| **总分** | 100% | | **5.65** | | **7.35** |

> 注：权重和分数应由你自己填，上面只是示例值。权重改变，结论可能改变。

### 比较完成后问自己

- [ ] 权重真的反映了我最在意的事情吗？
- [ ] 有没有某个依据被我忽视了，但其实很重要？
- [ ] 如果结论和直觉相反，是直觉有盲区，还是权重设错了？

## 第五步：时间维度与迭代

> 核心：任何重大决定都不是一次性的。用时间轴检查，用贝叶斯方式持续更新。

### 时间轴检查

| 时间段 | 选项A的影响 | 选项B的影响 | 哪个更好？ |
|--------|-----------|-----------|---------|
| 短期（1-2年） | | | |
| 中期（3-5年） | | | |
| 长期（5年以上） | | | |

**关键问题：三个时间段的结论一致吗？如果不一致，以哪个为主？**

### 示例

| 时间段 | 去美国 | 留加拿大 |
|--------|--------|---------|
| 短期（适应期） | 新环境适应成本高，语言文化再调整 | 已适应，相对稳定 |
| 中期（大学期间） | 可能接触不同资源和思维方式 | 同样可以，且经济压力更小 |
| 长期（毕业后） | 留美难，回加拿大需重建；但若心理/艺术走学术路线影响较小 | 留加路径更清晰；申请北美研究生同等竞争力 |

### 贝叶斯迭代：这个决定什么时候需要重新评估？

重大决定不是做一次就完，而是随着新信息不断更新。设定几个触发重新评估的节点：

```
触发点 1：[时间/事件] → 如果发生[X]，重新评估
触发点 2：[时间/事件] → 如果发现[Y]，重新评估
触发点 3：[时间/事件] → 如果条件[Z]改变，重新评估
```

### 示例

```
触发点 1：孩子17岁时 → 如果孩子对方向更清晰（如确定走心理学学术路线），
          重新评估美国顶尖研究型大学是否更有价值

触发点 2：申请季开始前 → 如果美国学校给出奖学金，经济差距缩小，
          重新评估成本权重

触发点 3：孩子自己表达明确意愿 → 如果孩子真正研究过目标学校和专业，
          而不只是跟随同学，动机变扎实，提高"孩子意愿"权重
```

## 最终决策输出

完成以上五步后，填写这张表：

```
【决定的定义】
这个决定实际上是：_______________

【最重要的因素】（来自第三步因果分析）
影响结果最关键的变量是：_______________

【比较结论】（来自第四步）
基于当前信息，[选项A / 选项B] 的加权得分更高（__ vs __）

【时间维度是否一致】
短中长期结论：[一致 / 不一致，以___期为主]

【当前结论】
在现有信息下，倾向于：_______________
原因：_______________

【还缺少哪些信息才能更确定】
1. _______________
2. _______________

【下次重新评估的时间/触发条件】
_______________
```

## 示例：完整填写版

```
【决定的定义】
这个决定实际上是：相比留在加拿大，去美国读大学是否能
更有效帮助对人文/心理/艺术感兴趣的孩子，在任何地方
获得有竞争力的职业发展。

【最重要的因素】
在人文/心理/艺术领域，决定"好工作"的关键不是本科在
哪个国家，而是孩子对方向的清晰度、研究生院校选择、
以及个人能力积累。

【比较结论】
基于示例权重，留加拿大得分更高（7.35 vs 5.65）

【时间维度】
短中长期结论基本一致：留加拿大路径更清晰、成本更低、
灵活性更高。

【当前结论】
在现有信息下，倾向于：留在加拿大完成大学，
除非孩子对美国某所具体学校的具体项目有真正的、
自己研究过的理由。

原因：动机来自从众；领域不强依赖美国品牌；
     成本差距巨大；签证风险高。

【还缺少的信息】
1. 孩子对未来方向是否有更具体的想法（心理学学术？艺术创作？）
2. 孩子是否真正研究过美国哪所学校、哪个项目

【下次重新评估】
孩子17岁，方向更清晰时；或申请季开始时
```
